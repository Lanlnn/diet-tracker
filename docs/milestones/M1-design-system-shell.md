# M1：设计系统与应用骨架

**状态：** 已完成
**依赖：** M0
**Git 分支：** `codex/m1-design-system`
**阶段标签：** `m1-complete`

## UI 基准

- `design/优化版/01` 至 `08` 的共性视觉。
- 暖米白背景、深苔绿主色、低饱和橙黄辅助色。
- 卡片圆角、弱阴影、4 级文字层级和底部五项导航。

## 目标

先建立稳定的视觉 Token、公共组件和五个一级路由骨架，后续页面只组合组件，不重复手写近似样式。

## 小程序任务

- 定义颜色、字号、字重、间距、圆角、阴影、安全区 Token。
- 建立五项 TabBar：今日、趋势、记录、运动、我的。
- 创建五个一级页面空壳和二级分包规划。
- 建立按钮、卡片、分段选择、状态页、营养进度条、列表行等组件。
- 处理 iPhone 刘海、底部安全区域和常见 Android 尺寸。
- 建立图标命名、尺寸和激活态规则。

## 后端任务

- 无新增业务接口。
- 提供前端本地 Mock 契约或测试数据结构。

## 测试

- 组件正常、禁用、按下、超长文本状态。
- 五个 Tab 可切换且中央记录按钮点击区域正确。
- 320px 至常见大屏宽度无横向溢出。
- 深色系统设置下仍按产品浅色主题稳定显示。

## 验收

- [x] 五项导航与 8 张优化稿保持一致。
- [x] 页面不再复制卡片、按钮和标题的重复 WXSS。
- [x] 主要点击区域不小于 44×44px。
- [x] UI Token 有示例页或组件预览页。
- [x] 旧版荧光绿色和旧 TabBar 已移除。
