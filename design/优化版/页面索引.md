# 优化版页面索引

## 一级页面

- 今日 → `01-首页-热量总览.png`
- 趋势 → `04-热量与运动趋势.png`
- 记录 → `05-食品搜索与记录.png`
- 运动 → `03-今日运动与推荐.png`
- 我的 → `08-个人中心.png`

## 核心操作

- 首页“记录一餐” → `05-食品搜索与记录.png`
- 选择食品 → `02-热量计算.png`
- 加入餐次／点击餐次 → `07-餐次详情.png`
- 首页日历 → `06-饮食日历.png`

## 收纳原则

- 食品搜索、拍照留存与自定义食物统一收进“记录”入口。
- AI 教练作为饮食或运动建议出现，不单独占一级页面。
- 训练计划从“运动”和“我的”进入。
- 教练、会员码、通知、隐私、关于均归入个人中心的低频设置。
